## 1. 快速入门

### 1.1 欢迎使用 NocoBase

NocoBase 是一款强大的无代码开发平台，让你无需编写代码就能构建复杂的企业应用。

### 1.2 在 5 分钟内创建第一个应用

#### 步骤 1: 登录系统

```
访问地址: http://localhost:13000
默认账号: admin@nocobase.com / admin123
```

#### 步骤 2: 创建数据表

1. 点击顶部菜单的 **"配置"** 按钮进入配置模式
2. 在左侧菜单选择 **"数据表管理"**
3. 点击 **"+ 新建数据表"**
4. 输入表名：`tasks`（任务）
5. 添加字段：
   - `title` (字符串) - 任务标题
   - `description` (文本) - 任务描述
   - `status` (选择) - 状态（待处理/进行中/已完成）
   - `priority` (选择) - 优先级（低/中/高）
   - `due_date` (日期) - 截止日期
6. 点击 **"保存"**

#### 步骤 3: 创建界面

1. 在左侧菜单选择 **"页面管理"**
2. 点击 **"+ 新建页面"**
3. 选择 **"空白页面"**
4. 从组件库拖拽 **"表格区块"** 到页面
5. 在右侧配置面板选择数据源：`tasks`
6. 配置显示字段：title, status, priority, due_date
7. 点击 **"保存"**

#### 步骤 4: 配置操作

1. 在表格区块的操作栏点击 **"+ 添加操作"**
2. 选择 **"添加新建"** - 允许创建新记录
3. 选择 **"添加编辑"** - 允许编辑现有记录
4. 选择 **"添加删除"** - 允许删除记录
5. 点击 **"保存"**

#### 步骤 5: 预览和使用

1. 点击顶部菜单的 **"使用"** 按钮退出配置模式
2. 现在你可以：
   - 查看任务列表
   - 创建新任务
   - 编辑和删除任务

恭喜！你已经创建了第一个应用。

## 2. 系统安装

### 2.1 系统要求

| 组件 | 最低要求 | 推荐配置 |
|------|---------|---------|
| **CPU** | 2 核 | 4 核+ |
| **内存** | 4GB | 8GB+ |
| **磁盘** | 20GB | 50GB+ SSD |
| **操作系统** | Linux/macOS/Windows | Linux |
| **Node.js** | 18.0+ | 20.14+ |
| **数据库** | PostgreSQL 12+ | PostgreSQL 14+ |
| **Redis** | 6.0+ | 7.0+ |

### 2.2 Docker 安装（推荐）

#### 2.2.1 使用 Docker Compose

1. 下载 `docker-compose.yml`:

```bash
wget https://github.com/nocobase/nocobase/releases/latest/download/docker-compose.yml
```

2. 启动服务:

```bash
docker-compose up -d
```

3. 访问系统:

```
URL: http://localhost:13000
初始账号: admin@nocobase.com / admin123
```

### 2.3 源码安装

#### 2.3.1 环境准备

```bash
# 克隆仓库
git clone https://github.com/nocobase/nocobase.git
cd nocobase

# 安装依赖
yarn install

# 复制环境变量文件
cp .env.example .env
```

#### 2.3.2 配置数据库

编辑 `.env` 文件：

```bash
# 数据库配置
DB_HOST=localhost
DB_PORT=5432
DB_DATABASE=nocobase
DB_USER=nocobase
DB_PASSWORD=nocobase

# Redis 配置
REDIS_HOST=localhost
REDIS_PORT=6379
```

#### 2.3.3 初始化数据库

```bash
yarn nocobase init
```

#### 2.3.4 启动开发服务器

```bash
# 开发模式
yarn dev

# 生产模式
yarn build
yarn start
```

### 3.2 配置模式 vs 使用模式

NocoBase 有两种模式：

#### 配置模式
- **用途**: 设计和配置应用
- **功能**: 创建数据表、设计界面、配置权限
- **入口**: 点击顶部 **"配置"** 按钮
- **视觉提示**: 页面显示配置边框和编辑按钮

#### 使用模式
- **用途**: 日常使用应用
- **功能**: 查看、创建、编辑数据
- **入口**: 点击顶部 **"使用"** 按钮
- **视觉提示**: 干净的用户界面

### 3.3 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl/Cmd + S` | 保存当前配置 |
| `Ctrl/Cmd + Z` | 撤销操作 |
| `Esc` | 退出当前编辑模式 |
| `Ctrl/Cmd + /` | 打开快捷命令面板 |

## 4. 数据管理

### 4.1 创建数据表

#### 4.1.1 数据表基础

数据表是存储数据的容器，每个数据表对应数据库中的一张表。

#### 4.1.2 创建步骤

1. 进入 **配置模式**
2. 点击 **"数据表管理"**
3. 点击 **"+ 新建数据表"**
4. 填写基本信息：
   - **表名称**: 数据表的唯一标识（英文，如 `orders`）
   - **显示名称**: 表的显示名称（中文，如 `订单`）
   - **描述**: 表的用途说明

#### 4.1.3 添加字段

支持的字段类型：

| 字段类型 | 说明 | 示例 |
|---------|------|------|
| **字符串** | 短文本，最多 255 字符 | 姓名、电话 |
| **文本** | 长文本，无长度限制 | 备注、描述 |
| **数字** | 整数或小数 | 年龄、金额 |
| **百分比** | 百分比值 | 折扣率 |
| **布尔** | 是/否 | 是否激活 |
| **日期** | 日期和时间 | 创建时间、生日 |
| **选择** | 预定义选项 | 状态、类型 |
| **多选** | 多个预定义选项 | 标签 |
| **文件** | 文件上传 | 附件、图片 |
| **图片** | 图片上传 | 头像、产品图 |
| **关联** | 关联其他表 | 用户、订单 |

### 4.2 配置表关联

#### 4.2.1 关联类型

| 关联类型 | 说明 | 示例 |
|---------|------|------|
| **一对一** | 两条记录一一对应 | 用户 - 个人资料 |
| **一对多** | 一条记录关联多条记录 | 订单 - 订单项 |
| **多对多** | 多条记录互相关联 | 商品 - 分类 |

## 6. 权限管理

### 6.1 用户管理

#### 6.1.1 创建用户

1. 进入 **设置 > 用户**
2. 点击 **"+ 新建用户"**
3. 填写用户信息：
   - **用户名**: 登录用户名
   - **邮箱**: 登录邮箱
   - **密码**: 初始密码
   - **角色**: 分配角色
4. 点击 **"保存"**

### 6.2 角色管理

#### 6.2.1 创建角色

1. 进入 **设置 > 权限 > 角色**
2. 点击 **"+ 新建角色"**
3. 输入角色信息：
   - **角色名称**: 角色标识
   - **显示名称**: 角色显示名称
   - **描述**: 角色说明
4. 点击 **"保存"**

### 6.3 权限配置

#### 6.3.1 资源权限

控制用户对数据资源的访问：

1. 进入 **设置 > 权限 > 角色**
2. 选择角色
3. 在 **"操作权限"** 标签：
   - 勾选允许的资源操作
   - 配置操作访问策略

## 7. 工作流

### 7.1 工作流概述

工作流引擎支持可视化配置业务流程，实现自动化处理。

### 7.2 创建工作流

#### 7.2.1 基本步骤

1. 进入 **设置 > 工作流**
2. 点击 **"+ 新建工作流"**
3. 填写基本信息：
   - **工作流名称**: 流程标识
   - **触发方式**: 选择触发器
4. 点击 **"确定"** 进入设计器

### 7.3 触发器配置

#### 7.3.1 触发器类型

| 触发器 | 说明 | 配置 |
|------|------|------|
| **定时触发** | 按时间计划执行 | Cron 表达式 |
| **事件触发** | 数据变更时执行 | 选择数据表和操作 |
| **手动触发** | 用户手动执行 | 添加到操作按钮 |

## 8. AI 集成

### 8.1 AI 助手配置

#### 8.1.1 创建 AI 助手

1. 进入 **设置 > AI 助手**
2. 点击 **"+ 新建助手"**
3. 配置助手信息：

#### 8.2.1 添加 AI 节点

1. 在工作流中添加 **"AI 调用"** 节点
2. 配置 AI 参数：
   - **选择助手**: 选择配置的 AI 助手

## 10. 系统设置

### 10.1 基础设置

#### 10.1.1 系统信息

1. 进入 **设置 > 系统 > 基本信息**
2. 配置系统信息：
   - **系统名称**: 应用名称
   - **Logo**: 上传系统 Logo
   - **主题色**: 界面主题颜色

### 10.2 邮件配置

#### 10.2.1 SMTP 配置

1. 进入 **设置 > 系统 > 邮件**
2. 配置 SMTP 参数：
   - **主机**: SMTP 服务器地址
   - **端口**: 端口号（通常 465/587）
   - **用户名**: 邮箱账号
   - **密码**: 邮箱密码或授权码

3. 点击 **"发送测试邮件"** 验证配置

## 11. 常见问题

### 11.1 安装和配置

**Q: Docker 启动失败怎么办？**

A: 检查以下几点：
1. 确认 Docker 已正确安装并运行
2. 检查端口 13000 是否被占用
3. 查看容器日志：`docker-compose logs -f`

**Q: 如何重置管理员密码？**

A: 使用 CLI 命令：
```bash
yarn nocobase reset-password
```

### 11.2 数据操作

**Q: 如何批量导入数据？**

A:
1. 准备 CSV 或 Excel 文件
2. 在数据表中点击 **"导入"**
3. 配置字段映射
4. 执行导入
